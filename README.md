# 📱 App UniShop  
**แอป e-commerce บน Mobile**

**UniShop** คือแอปพลิเคชัน e-commerce สำหรับขายสินค้าหลากหลายประเภท เช่น  
- 📱 โทรศัพท์มือถือ  
- 💻 โน้ตบุ๊ค  
- ⌚ นาฬิกา  
- 👕 เสื้อผ้า  
- 👟 รองเท้า  
- 💄 เครื่องสำอาง ฯลฯ

โปรเจ็กต์นี้แบ่งเป็น 2 แอป ได้แก่:

## 🔵 1. User App  
แอปสำหรับผู้ใช้งานทั่วไป เลือกซื้อสินค้าได้หลายหมวดหมู่
